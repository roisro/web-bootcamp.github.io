Penjelasan Program Full Stack

Program Full Stack JavaScript terdiri dari fase Preparation sampai dengan Bootcamp yang terdiri dari 3 Fase

Preparation

Pada Fase 0 siswa akan mempelajari dasar-dasar dan pola pikir pemrograman yang akan menjadi bekal untuk fase berikutnya. Akan ada one-on-one feedback untuk memfasilitasi murid-murid.

Fase 1

Pada Fase 1 siswa akan mempelajari lebih lanjut tentang application structure, pengenalan database, ORM (Object Rational Mapping) dan Paradigma OOP (Object-Oriented Programming).

Fase 2

Pada Fase 2 siswa akan diajarkan membuat beragam portofolio sebagai web developer, belajar mengenai project management dan paradigma functional programming.

Fase 3

Mengimplementasikan React & react Native, mendalami web application scaling dan deployment, membangun MVP (Minimum Viable Product) dan membuat Final Project.
